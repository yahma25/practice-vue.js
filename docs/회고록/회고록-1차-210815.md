## 느낀점

### 테스트 환경 설정

환경 설정은 항상 어렵게 느껴진다. 특히 OS에 따른 전용 환경 설정은 더 까다롭다. (~~나 빼고 다 mac만 쓰나봐~~) 디버깅을 할 수 없어서 더 답답함이 느껴졌다.

### 테스트코드 관련 글 읽기

테스트(TDD 포함)에 대해 깊게 생각해오지 않아서 그런지 모든 것이 흥미로웠다. 앞으로 더 많이 글도 읽고, 기술도 익히고, 기록도 남겨볼 예정이다.
